# Bulk Email Platform

Modern bulk email sender using:

- React + Vite
- Tailwind CSS
- Node.js + Express
- Nodemailer
- Gmail SMTP
- Excel/CSV Upload

## Features

- Upload Excel/CSV
- Personalized Emails
- Modern Dashboard UI
- Gmail App Password Support
- Bulk Email Sending

## Setup

### Install

```bash
npm install
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

## Environment Variables

Create `.env` inside server:

```env
PORT=5000
EMAIL_USER=yourgmail@gmail.com
EMAIL_PASS=your_gmail_app_password
```
